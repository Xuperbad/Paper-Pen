## 时序图示例

```mermaid
sequenceDiagram
    participant 读者
    participant 网站
    participant 书籍内容
    
    读者->>网站: 访问Paper-Pen
    网站->>读者: 显示书籍列表
    读者->>网站: 选择《南明史》
    网站->>书籍内容: 加载章节
    书籍内容->>网站: 返回内容
    网站->>读者: 显示章节内容
```

## 类图示例 - 书籍分类

```mermaid
classDiagram
    class 书籍 {
        +String 标题
        +String 作者
        +Date 出版日期
        +int 章节数
        +阅读()
        +做笔记()
    }
    
    class 历史书籍 {
        +String 历史时期
        +String 地理范围
        +分析历史事件()
    }
    
    class 科技书籍 {
        +String 技术领域
        +String 应用场景
        +学习新概念()
    }
    
    书籍 <|-- 历史书籍
    书籍 <|-- 科技书籍
    
    历史书籍 : 南明史
    历史书籍 : 中国通史
    科技书籍 : 游戏改变世界
```
